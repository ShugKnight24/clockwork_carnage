# Clockwork Carnage — Episode 1: The Alpha Protocol

A retro-style first-person shooter built entirely with vanilla JavaScript, HTML5 Canvas, and procedural art. No frameworks. No sprites. Every pixel drawn in code.

## Play

Open `index.html` in a modern browser. No build step required.

## Features

**Campaign Mode** — 3-act story across 9 levels with cutscenes, a comic book origin sequence, and a multi-form boss fight. Meet ARIA, your AI companion who provides tactical callouts, idle chatter, and personality.

**Arena Mode** — Endless wave survival with 4 difficulty settings, 18 upgrades, kill streaks (DOUBLE KILL → GODLIKE), slow-mo last kills, and stats tracking.

**Builder Mode (Temporal Forge)** — Design your own levels with a multi-layer editor. Undo/redo, export/import as JSON, multi-map save slots, spawn point placement, and play-test your creations.

**Character Creator** — Customize across 5 categories with loadout bonuses that affect gameplay.

**Accessibility** — Colorblind mode, font scaling, mobile touch controls, orientation lock.

## Controls

| Action      | Key             |
| ----------- | --------------- |
| Move        | WASD            |
| Look        | Mouse           |
| Shoot       | Left Click      |
| Sprint      | Shift           |
| Dash        | Double-tap WASD |
| Interact    | E               |
| Weapons 1-4 | 1-4             |
| Pause       | ESC / P         |
| Toggle FPS  | F               |

All controls are rebindable in Settings → Controls.

**Builder:** Arrow keys to move, Q/E to switch layers, Space to place, X to remove, T for spawn points, Ctrl+Z/Ctrl+Shift+Z for undo/redo, Ctrl+E/I for export/import.

**Mobile:** Full touch control support with virtual joystick and action buttons.
